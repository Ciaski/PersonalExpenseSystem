# PersonalExpenseSystem 

Questo progetto è un sistema software per la gestione delle spese personali, sviluppato per integrare un linguaggio di programmazione (Python) con un database relazionale (SQLite).

##  Requisiti Tecnici
- **Linguaggio:** Python 3.x
- **Database:** SQLite3
- **Librerie standard:** `sqlite3`, `datetime`

##  Struttura del Progetto
- `src/`: Contiene il codice sorgente Python (`main.py`).
- `sql/`: Script per la creazione delle tabelle e dati di esempio.
- `demo/`: Video dimostrativo del funzionamento del software.

## Istruzioni per l'Installazione e Avvio
1. Assicurati di avere Python installato sul tuo sistema.
2. Clona il repository o scarica i file.
3. Apri il terminale nella cartella del progetto.
4. Esegui il comando:
   ```bash
   python src/main.py
